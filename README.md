# Sales and Profit Insights Dashboard

Power BI Internship Project 

This project is a Power BI dashboard designed to analyze sales and profit performance
across different business segments, countries, and years.

## Project Context
This dashboard was developed as part of the **Microsoft Elevate Power BI Internship Program**
for learning and academic purposes.

## Tools Used
- Microsoft Power BI Desktop

## Dataset
- Power BI Financials Sample Dataset

## Dashboard Features
- KPI cards showing Total Sales, Total Profit, Units Sold, and Profit Margin
- Sales performance analysis across segments
- Profit contribution by segment
- Country-wise sales comparison
- Profit distribution across countries
- Year-wise sales trend and sales vs profit comparison
- Interactive cross-filtering between visuals

## Output
- Dashboard screenshots are available in the `screenshots` folder
- High-quality PDF export of the dashboard is included
